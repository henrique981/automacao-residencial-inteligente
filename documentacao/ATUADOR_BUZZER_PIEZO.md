# Documentacao Tecnica - Buzzer Piezo
Data: 2026-06-24
Status: Completo

## Componentes
- Buzzer Piezo (D5) - Gerador de Sons
- Resistor 220 ohm - Protecao

## Pinos
- + (positivo) = D5 (via resistor 220 ohm)
- - (negativo) = GND

## Frequencias
- 200 Hz = Som grave
- 440 Hz = La padrao (musical)
- 1000 Hz = Som medio
- 2000 Hz = Som agudo

## Notas Musicais
- Do (C4) = 262 Hz
- Re (D4) = 294 Hz
- Mi (E4) = 330 Hz
- Fa (F4) = 349 Hz
- Sol (G4) = 392 Hz
- La (A4) = 440 Hz
- Si (B4) = 494 Hz

## Testes
- Beep simples OK
- Escala musical OK
- Alerta frequencia alta OK
- Padrao morse OK
- Transicao grave agudo OK
- Duplo confirmacao OK

## Aplicacoes
- Alarme de gas (cozinha)
- Notificacao de presenca
- Alerta de temperatura
- Confirmacao de acoes
