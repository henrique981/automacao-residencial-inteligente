# Automacao Residencial Inteligente com IoT

Sistema completo de automacao residencial para apartamento 70m2 usando ESP32, sensores IoT e Home Assistant.

## Estrutura do Projeto
## Projetos Wokwi

- Sala Inteligente (DHT22, PIR, LDR, Rele)
- Cozinha Segura (MQ-2, LDR, Rele)
- HC-SR04 Ultrassonico
- LED RGB (16.7 milhoes de cores)
- Servo SG90 (0-180 graus)
- Buzzer Piezo (sons e frequencias)
